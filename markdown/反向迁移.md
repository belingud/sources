```shell
flask-sqlacodegen 'mysql+pymysql://user:password@localhost/database' --outfile models.py 
--flask
```

这是将整个数据库生成model

```shell
flask-sqlacodegen 
'mysql+pymysql://username:password@127.0.0.1/database' 
--table table 
--outfile model.py
--flask
```
生成指定表的model
