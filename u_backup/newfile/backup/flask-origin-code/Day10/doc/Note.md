### 函数

- 对象函数
  - 对象调用
  - 可以使用类中的所有资源
- @classmethod  类函数
  - 类调用
  - 对象调用
  - 可以使用类资源
  - 不能使用对象资源
- @staticmethod  静态函数
  - 对象调用
  - 类调用
  - 类和对象的资源都不能调用
  - 相当于寄生在我们的类上
